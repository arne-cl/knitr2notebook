knitr2notebook
==============

knitr2notebook converts knitr documents (R markdown *.Rmd files) to
Ipython notebooks. The code is based on https://gist.github.com/ramnathv/9334834 .

Usage
-----

::

    knitr2notebook input.Rmd output.ipynb

Dependencies
------------

- R
- knitr
- rpy2
- notedown
